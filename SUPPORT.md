# Support

This project is provided as a sample for demonstration and learning purposes.

## Support Scope

- Documentation improvements
- Bug fixes that keep the sample runnable
- Reasonable clarifications about setup and usage

## Not Included

- Production support
- SLA-backed response times
- Custom feature development
- Deployment reviews for regulated environments

## Getting Help

- Open a GitHub issue for bugs or documentation gaps
- Review [README.md](README.md) for setup instructions and architecture notes
- Review [SECURITY.md](SECURITY.md) for vulnerability reporting guidance
